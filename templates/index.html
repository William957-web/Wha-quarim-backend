API LIST

If you found any bug on this program. plz must report it to the dev.

/api/reg/username/:username:/password/:pwd:


Registration, for example : /api/reg/username/whale120/password/password1234

And it would return a random token(sha256 hashed), for example : 26a989a23abf5ee3ae4106883e9f534a8042846458e34df03ab5ee063692f007

/api/login/username/:username:/password/:pwd:


Login, for example : /api/login/username/whale120/password/password1234

And it would return a random token(sha256 hashed), for example : 26a989a23abf5ee3ae4106883e9f534a8042846458e34df03ab5ee063692f007

/api/userlogout/:token:


Logout with user token, and you have to login again to get a new token

/api/getUserdata/token/:token:


Getting JSON format user data, for example http://192.168.36.129/api/getUserdata/token/26a989a23abf5ee3ae4106883e9f534a8042846458e34df03ab5ee063692f007

Example return data : {"userdata": [[1, "meow", "61be55a8e2f6b4e172338bddf184d6dbee29c98853e0a0485ecee7f27b9af0b4", "{}", "{}", 100, 0, "{}", "{}", "26a989a23abf5ee3ae4106883e9f534a8042846458e34df03ab5ee063692f007"]]}

id, username, password, fishes data, foods data, money data, total play duration time, pre_fishes, acheivements, token
